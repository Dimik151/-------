# Различные задачи


task_1.php 
Напишите функцию, которая принимает последовательность слов и проверяет, могут ли слова быть организованы в одну непрерывную цепочку слов. 
Пример: cat -> cot -> coat -> oat -> hat -> hot -> hog -> dog
Расстояние Левенштейна
